AI Tools Directory

2,800+ records · Free sample (30 records, JSON + CSV) · Full dataset $9 · Sample refreshed 2026-05-24

Catalog of 2,800+ AI tools and applications with category, pricing model, and API availability.

What you'd use this for

  1. AI tool comparison content site / SEO play — 2,700+ tools across 10 clean categories (text, code, image, audio, video, multimodal, etc.). Build an SEO comparison engine ("Top 20 AI image tools 2026"). Each row already has pricing + API flag — saves manual research.
  2. Internal "approved AI tools" catalog for an enterprise — A 200-person company's IT team filters by pricing=open-source (1,124 records) and hasApi=true (506 records) → an internal catalog of self-deployable, programmatic AI tools. No vendor lock-in scare.
  3. AI tool recommendation engine for an LLM agent — Embed the dataset → your agent gets "what AI tool should I use for {{task}}". 699 text tools, 811 code tools, 318 image tools — agent picks the right one based on user intent + free/paid filter.
  4. Newsletter / content series sourcing — A weekly "5 underrated AI tools you missed" newsletter — sourcing from the long-tail categories (multimodal: 69, other: 14) where competition for attention is low.
  5. Lead-gen for AI consultancy — The dataset's tag breakdown ("large language models": 271, "developer tools": 53, "image synthesis": 113) is a vertical-mapping cheat sheet. AI consultants use it to identify which AI vendors a prospect is likely using.
