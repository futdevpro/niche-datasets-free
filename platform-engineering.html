Platform Engineering & IDP Tooling Directory

390+ records · Free sample (20 records, JSON + CSV) · Full dataset $11 · Sample refreshed 2026-05-13

Catalog of 390+ platform engineering tools across developer portals, GitOps, service mesh, IaC, observability, container orchestration, CI/CD, secrets, policy, feature flags, service catalog, cost management, and DX.

What you'd use this for

  1. Stack selection for a new internal developer platform — Engineering org spinning up a "Platform Engineering" team needs to pick the foundational tools. Filter dataset by category=developer-portal → see Backstage / Port / Cortex / OpsLevel / Humanitec ranked by license + maturity. Filter by category=gitops → ArgoCD vs Flux comparison. Filter by category=service-mesh → Istio vs Linkerd vs Kuma. Saves the "evaluate 50 tools to pick 5" research week.
  2. CNCF landscape consolidation — The CNCF landscape page is overwhelming (500+ icons across 30+ categories). Buyers download this dataset, filter by their relevant categories, and get a 10-row shortlist instead of a 500-icon visual. Cross-reference license tier (permissive vs copyleft) and the result is a procurement-ready vendor list.
  3. SRE / Platform Engineer career mapping — SRE moving into a platform-engineering role uses the dataset to map "what tools should I learn?" by category. Filter category=observability → Prometheus + Grafana + OpenTelemetry + Datadog + Honeycomb. Each becomes a learning track. Cross-reference popularity (GitHub stars proxy via the urlDomainTier=repo-hosted records) to prioritize the most-deployed tools first.
  4. IDP RFP / vendor evaluation — Enterprise tech-PM evaluating Backstage vs Port vs Cortex vs OpsLevel for an internal-platform RFP. Filter category=developer-portal → all 5 vendors present with description + URL. Pair with the categorizer's keyword fallback (recognizing "internal developer platform" and similar terminology) to surface adjacent tools (Coherence, Qovery, Humanitec) the buyer didn't know existed.
  5. Comparison content / SEO — "Argo CD vs Flux: which GitOps engine?" / "Backstage vs Port vs Cortex: which IDP?" / "Istio vs Linkerd: service mesh tradeoff" comparison content series. Each category in the dataset becomes an evergreen comparison article. Drives organic search from buyer-intent queries.
  6. Internal tooling consolidation audit — Big-co platform team auditing their internal tools (Are we using too many CI/CD systems? Do we have 3 secrets managers?) downloads the dataset to benchmark "what's the standard" per category. Filter category=secrets-management → see Vault / AWS Secrets Manager / Sealed Secrets / Doppler / sops as the canonical options.
  7. Cross-sell into Dev Stack pack — Pair this dataset with developer-tools (broader OSS dev tools), npm-packages (JS/TS ecosystem), homebrew-packages (macOS CLI tooling), vscode-extensions (IDE layer), and self-hosted-software (broader OSS) → the complete platform-team toolkit from CLI to portal to runtime. Sub-bundle "Platform Builder Pack" $25-29 viable.
