AI Agents Directory

2,000+ records · Free sample (20 records, JSON + CSV) · Full dataset $7 · Sample refreshed 2026-05-13

Catalog of 2,000+ AI agent projects and frameworks — autonomous agents, multi-agent systems, agent-as-a-service tools.

What you'd use this for

  1. AI agent framework comparison engine — 1,324 frameworks in the dataset. Build a "Pick the right agent framework" comparison site (LangChain vs CrewAI vs Autogen vs LlamaIndex Agents vs SmolAgents, etc.) with structured data per row.
  2. Coding agent shopping list — 134 coding-focused agents (Cursor, Cline, Aider, OpenHands, Devin, etc.). Engineering teams comparing options can use the dataset as a structured spec sheet — instead of 30 separate landing-page reads.
  3. Research agent / academic mapping — 75 research agents (autonomous deep-research bots, paper summarizers, etc.). Academics + market intel teams use the dataset to track the autonomous-research space.
  4. "Build vs. adopt" decision support — Product teams asking "should we build our own agent or adopt {{framework}}?" — the dataset's category=framework filter (1,324 records) shows the OSS option-space, with pricing showing the build-or-buy economics.
  5. AI-agent registry for an LLM playground — Embed the dataset → an LLM playground / IDE plugin that recommends agents based on user intent. "I want to scrape a website" → returns matching scraping agents from the dataset.
