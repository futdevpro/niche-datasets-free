Catalog of 4,000 most-downloaded HuggingFace models with downloads, likes, license, pipeline tag, modality, and language.

What you'd use this for

  1. Self-hosted LLM picker for a privacy-sensitive deployment — Healthcare / legal / finance customer can't send data to commercial APIs. Filter the dataset by pipeline_tag=text-generation + license=apache-2.0 + sort by downloads → top-vetted self-hostable LLMs with permissive licenses. Ship the picker as a config option in your app.
  2. "Build vs. buy" cost calculator — Pair this dataset with ai-models-pricing (commercial APIs). For each model in your shortlist, compute "API cost per million tokens at our usage volume" vs. "GPU cost per million tokens self-hosted at our utilization". The breakeven point tells you when to switch from rent → own.
  3. Embedding model selection for a RAG pipeline — RAG needs an embedding model. Filter dataset by pipeline_tag=feature-extraction OR pipeline_tag=sentence-similarity → 400+ embedding models. Sort by downloads → sentence-transformers/all-MiniLM-L6-v2 (247M downloads) is the safe baseline. The dataset shows the alternatives if you need bigger context (BAAI/bge-large) or multilingual (paraphrase-multilingual).
  4. Image-classification baseline for an MVP — Need image classification but don't want to train a model? Filter by pipeline_tag=image-classification + sort by downloads → top is timm/mobilenetv3_small_100 (22M downloads). Ship the MVP with a battle-tested 1MB model instead of fine-tuning.
  5. License-compliance audit for an existing pipeline — ML pipeline already uses 5-10 HF models. Each model's license is buried in the README. Cross-reference your model IDs with this dataset's license field → flagged in 30 seconds: "model X is gpl-3.0, will block commercial deployment".
  6. Library-preference filter for engineering teams — "We standardized on PyTorch" → filter library_name=transformers (excludes mlx / tflite / onnx forks). "We're on Apple Silicon" → filter library_name=mlx. The dataset surfaces the lib boundary that's invisible on the HF UI.
  7. Trend analysis: what's growing in open-weight ML? — createdAt + downloads enable cohort analysis. Filter to "models created in last 6 months" → see which categories the community is actively forking. This quarter's answer: multimodal + reasoning + MLX-targeted models.
