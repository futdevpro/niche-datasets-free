Developer Tools Directory

1,600+ records · Free sample (20 records, JSON + CSV) · Full dataset $9 · Sample refreshed 2026-05-20

Catalog of 1,600+ developer tools across CLI tools, code editors, debuggers, profilers, and productivity utilities.

What you'd use this for

  1. Internal "approved tools" catalog for an engineering org — A 100-person eng team filters by pricing (free / freemium / paid) + category (monitoring / mail / messaging / CI/CD), and ships an internal procurement-ready catalog. SOC2 audit asks "what tools are you using" — answer is one query away.
  2. Dev-productivity team's tooling-stack benchmark — Filter by category=ide-editor (93 records), automation (110), code-quality (66) and you have a structured comparison sheet for picking dev tools. Easier than crawling DevTools.fm episodes.
  3. Indie founder's launch-stack picker — Filter by pricing=free (585) + pricing=freemium (687). A solo founder building an MVP gets a 1,200+ record list of free-tier-friendly dev tools, sorted by category. Stack-decision in an afternoon.
  4. Content site: "Best free X tool for Y use-case" — 178 integrations, 178 infrastructure, 110 automation tools — endless niche-specific comparison content. Each row is a tool with pricing + API flag, ready to embed in a comparison table.
  5. Lead-gen for B2B dev SaaS — The dataset's vendors are also potential B2B customers and partners. A growth team filters hasApi=true (482 records) — these are companies with developer-facing APIs, mapping the API-economy ICP.
  6. AI agent's "what tool can I call" registry — An agentic LLM needs to know "what dev tools have APIs". Filter hasApi=true (482 records) → the agent knows which 482 tools it can integrate with programmatically, which is most of the time-saving value of using agents at all.
