Self-Hosted Software Directory

2,200+ records · Free sample (30 records, JSON + CSV) · Full dataset $14 · Sample refreshed 2026-05-24

Catalog of 2,300+ self-hostable open-source applications across communication, file transfer, smart home, and more.

What you'd use this for

  1. Build a "Self-host this instead of SaaS X" comparison engine — Match a SaaS service (Slack, Notion, Dropbox, etc.) to a self-hosted alternative from the dataset. Filter by category=communication (240 apps), cms (55 apps), file-transfer (123 apps). Output: blog SEO content, Chrome extension, or a paid alternative-finder service.
  2. Internal IT team's "approved app" catalog — A 50-person company's IT team imports the dataset to build a curated internal catalog of self-hostable tools, filtered for pricing=open-source (96% of records). Replaces ad-hoc Slack-message recommendations. Shipping time: 2 days instead of 2 months of research.
  3. Smart-home / home-lab dashboard — 89 smart-home apps + 91 media-streaming + 123 file-transfer = a complete homelab landing page. Use the dataset to power a personal dashboard or "What should I self-host this weekend?" recommendation engine.
  4. Privacy-focused product directory — Filter pricing=open-source AND category IN (communication, social-networks), then build a "Replace WhatsApp/Slack/Twitter with these privacy-first alternatives" content site. The dataset's 180-record nostr cluster alone makes a niche subsite.
  5. Lead-gen for a managed-hosting business — Sell managed hosting for popular self-hosted apps. The dataset gives you a launch list of which apps to support first (sorted by category-popularity), what they need (Docker, DBs, etc.), and a vertical to target (CMS, communication, finance).
