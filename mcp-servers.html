Open MCP Servers Directory

3,600+ records · Free sample (20 records, JSON + CSV) · Full dataset $9 · Sample refreshed 2026-05-13

Catalog of 3,600+ Model Context Protocol (MCP) servers with GitHub stars, category, and capabilities.

What you'd use this for

  1. Claude / Cursor / Cline integration sourcing — Every Claude Desktop, Claude Code, Cursor, Cline, etc. user shopping for MCP servers — 3,600+ records sorted by category. "What MCP server should I install for Postgres / GitHub / Slack / Notion / web search?" — answer is one filter away.
  2. Build an "MCP server registry" SaaS / open hub — The MCP ecosystem doesn't yet have a Spotlight-style 1-click registry. The dataset is the database — wrap it in a UI, charge for a hosted index, or open-source it as a Smithery competitor.
  3. Internal MCP catalog for an AI-using enterprise — A 100-person company adopting Claude / agents internally needs to know what MCPs are safe / vetted / category-ranked. The dataset filters by pricing=open-source (97% of records) → an internal "approved MCP" list.
  4. AI agent's tool-discovery layer — An agent that can dynamically choose MCPs at runtime needs a registry. Embed the dataset → agent's planner can semantic-search "give me an MCP that does {{X}}". 532 dev-tools MCPs, 258 AI-services MCPs, 336 finance MCPs.
  5. Lead-gen for MCP-as-a-service businesses — Anthropic's MCP partner program is forming. Companies running MCP servers (especially community servers: 227 records) are launch-ready B2B targets for tooling, monitoring, hosting, and security.
  6. "Awesome MCP" curation backbone — GitHub awesome-mcp-servers is one of the dataset's source lists — but messy. Use the structured version to power an actually-good "awesome list" alternative or curated newsletter.
