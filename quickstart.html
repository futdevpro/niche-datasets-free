5-Minute Quickstart

Goal: by the end of this page, you'll have downloaded a sample, inspected its schema in your editor, and decided whether to grab the full dataset. No signup, no API key.

1 Pick a dataset ~30 seconds

Scan the 20-dataset catalog. The "Records" column tells you the size of the full version; the "Sample" column links you to a 20-record JSON or CSV preview. Pick the one closest to what you're trying to build.

Don't know which to pick? Common starting points:

  • Building a dev-tools comparison page → npm-packages + homebrew-packages
  • Building an AI agent / MCP integration → mcp-servers + ai-agents
  • Pricing comparison across LLM providers → ai-models-pricing
  • HuggingFace ecosystem research → huggingface-models + huggingface-datasets
  • Internal "approved tools" catalog → developer-tools + vscode-extensions

2 Grab the sample ~10 seconds

From the dataset's detail page, click the JSON or CSV sample link. Or grab it directly via curl:

curl -O https://raw.githubusercontent.com/futdevpro/niche-datasets-free/main/npm-packages-sample.json
# or for CSV:
curl -O https://raw.githubusercontent.com/futdevpro/niche-datasets-free/main/npm-packages-sample.csv

The sample is 20 records. Same schema, structure, and quality as the full version — just a random subset.

3 Inspect the schema ~2 minutes

Open in your favorite tool:

# jq: print all field names + types
cat npm-packages-sample.json | jq '.[0] | to_entries | map({key: .key, type: (.value | type)})'

# Python: dump first record
python3 -c "import json; print(json.dumps(json.load(open('npm-packages-sample.json'))[0], indent=2))"

# Or just open the .csv in Excel / Google Sheets / Numbers

Confirm the records have the fields you need. Every record has the base schema (name, url, description, category, pricing, hasApi, tags) plus dataset-specific first-class fields. See the per-dataset detail page for the field reference.

4 Run a real query ~1 minute

Filter the sample to see if it answers a question you'd actually ask:

# jq: all open-source records with an API
jq '.[] | select(.pricing == "open-source" and .hasApi)' npm-packages-sample.json

# jq: top 5 by category count
jq '[.[].category] | group_by(.) | map({c: .[0], n: length}) | sort_by(.n) | reverse | .[:5]' npm-packages-sample.json

# Python pandas
python3 -c "
import pandas as pd
df = pd.read_csv('npm-packages-sample.csv')
print(df.groupby('pricing').size())
print(df.groupby('category').size().sort_values(ascending=False).head())
"

If the sample answers your question correctly, the full version (~20-12,200 records depending on dataset) will scale the same query without changes.

5 Decide ~30 seconds

The sample answered your question: grab the full dataset on Gumroad ($7-$14 per dataset, $24-$34 for themed bundles, $34 for the complete bundle of all 20). One-time payment, instant download. The detail page for the dataset you picked has the direct Gumroad link.
The sample didn't fit: check the other 19 datasets, or read vs. alternatives to see if Kaggle / HuggingFace / awesome-* / a custom scrape would be a better match. The samples are free — try as many as you want.
