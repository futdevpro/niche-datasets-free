How the data is built

Each dataset is sourced from official APIs or curated from multiple public open-source lists, then run through an automated pipeline: sources → crawl → normalize → dedupe → enrich → validate → export → publish. Refresh cadence: monthly for ai-models-pricing, quarterly for the rest. The samples here are random subsets of the full versions — same schema, structure, and quality.

Built by Ronnie J at Future Development Program. Free samples are MIT-style permissive for evaluation. Full datasets licensed per Gumroad terms.
