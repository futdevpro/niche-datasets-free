Design Resources Directory

2,100+ records · Free sample (30 records, JSON + CSV) · Full dataset $9 · Sample refreshed 2026-05-24

Catalog of 2,100+ design resources across design systems, UI kits, icons, illustrations, stock photos, and tools.

What you'd use this for

  1. Design system bootstrap kit — A team starting a new design system filters the dataset by category=design-systems (92 records) + pricing=open-source (478 across the dataset) and gets a curated shortlist of OSS design systems to study or fork. Skip the typical 2 days of "what's out there" research.
  2. Frontend dev's component-library shopping list — 236 UI component records + 105 CSS framework records. Filter by pricing=free or open-source, sort by category, and you have a structured comparison table for picking React/Vue UI libraries — saves the typical "10 GitHub repos open in tabs" research.
  3. Stock-photo / icon discovery for content teams — 126 stock photo sources + 190 icon resources, with pricing flag. Content teams pick the 5-10 sources that fit their license/budget and ditch shutterstock/getty subscriptions.
  4. SEO content site: "Best free design tools for X" — 2,166 records with categories (UI, icons, colors, stock photos, prototyping). Build niche-specific comparison content sites ("Best free icon libraries for SaaS dashboards") — each row is a row in your comparison table.
  5. Internal design-asset catalog for an agency — A 30-person design agency builds an internal catalog of approved design resources, filtered by license, type, and category. Onboarding new designers gets faster — they get the catalog instead of the senior's Slack history.
