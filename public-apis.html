Public APIs Directory

2,500+ records · Free sample (20 records, JSON + CSV) · Full dataset $12 · Sample refreshed 2026-05-13

Catalog of 2,500+ free and freemium public APIs with auth, HTTPS, and CORS flags per row, across 50+ categories.

What you'd use this for

  1. AI agent's tool-registry / function-calling backend — An agentic LLM needs a registry of "what APIs can I call". Filter the dataset to auth=no-auth (770 records) and cors=true (502 records) → the agent gets a list of APIs it can call from a browser without secrets management. Saves weeks of API integration plumbing.
  2. Hackathon / weekend-project starter — "What public API can I use this weekend?" — filter by category (finance, games-comics, cryptocurrency, weather) and pricing=free. The dataset becomes a 30-second project starter instead of a Google rabbit hole.
  3. Geocoding-API comparison engine — **163 geocoding APIs** in the dataset. Build a comparison microsite ("Best free geocoding API for {{country}}"), each row with auth, HTTPS, CORS flags. SEO content + affiliate revenue.
  4. Lead-gen for B2B fintech / cryptocurrency / open-data — 122 finance APIs, 76 cryptocurrency, 115 open-data — these are publishers and product owners. A growth team can use the dataset as a sourcing pipeline: vendors who maintain public APIs are usually B2B SaaS targets.
  5. "Free APIs that don't need a credit card" content site — 30.2% of all public APIs are entirely free (779 records). Content angle: "30 free APIs you can use today, no signup needed". The dataset is the table-of-contents.
  6. RAG / search index for AI tool builders — Embed the dataset into a vector store. Now your AI agent / IDE plugin can answer "find me a free API that returns ISO country codes with CORS support" by semantic search.
